# 影期 Release Checklist

## 当前定位
影期是为摄影师个人与摄影工作室准备的档期、客户、跟进、分工和回款管理工具。

## 正式版重点
- 今日工作台可直接看到我负责与团队其他安排
- 档期页支持按成员聚焦，多场同日拍摄更清楚
- 订单详情聚焦执行、分工、回款和同日排班
- 本地备份恢复与显式 iCloud 同步

## 上架前复核
- App Icon、启动图、品牌文案统一为“影期”
- 登录页、设置页、分享确认单文案一致
- 游客模式、登录模式、iCloud 同步路径清晰
- 大量档期与多成员分工下滚动和切换无明显卡顿
- 隐私说明与实际功能一致，不自动请求定位

## 建议回归场景
- 一天 3 到 4 场拍摄，检查“我负责 / 团队其他”是否清楚
- 新建订单后默认地点、备注、定金比例是否自动带入
- 订单详情里新增 / 编辑 / 删除回款是否准确
- 归档、恢复、删除订单是否同步更新相关列表
- 导出、恢复备份与示例数据导入是否稳定
