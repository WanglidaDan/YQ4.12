# 影期 App Store 上架文案模板

## App 名称
影期

## 副标题
摄影工作室的经营与协作中台

## 宣传文案
把档期、客户、合同、回款、资料与团队协作全部收进一个工作台。

## 描述
影期是一款为摄影师、摄影团队与摄影工作室设计的业务管理工具，帮助你把档期、客户、拍前确认、团队分工、回款记录和交付节点整理到同一个工作区。

### 核心能力
- 档期总览：按月、周、列表查看全部拍摄安排
- 客户管理：沉淀来源、阶段、层级与历史合作记录
- 跟进提醒：拍前确认、报价推进、交付回访集中管理
- 团队分工：按成员查看“我负责”和团队其他安排
- 回款管理：记录每笔付款流水，自动计算已收与待回款
- 经营中心：报价、合同、收据、发票闭环一站完成
- 外部日历整备：展示订单级接入说明与准备状态，当前正式版不默认开放系统日历 / Google Calendar 自动同步
- 资料中心：支持导入附件、参考图、合同附件与外部链接
- 团队权限：角色权限、成员管理、操作留痕和导出控制
- 经营分析：签约额、回款、待收、客单价、复购率、渠道表现
- 备份与恢复：支持 JSON / CSV 导出与完整备份（含附件）

### 适合谁
- 婚礼摄影师
- 个人写真 / 情侣 / 亲子摄影师
- 商业摄影团队
- 中小型摄影工作室主理人
